Contoh Sematan Widget di populicenter.org

Cukup tempel satu baris <script> di halaman mana pun. Ganti host API dengan URL produksi Anda saat live. 

<script
  src="https://API-HOST/api/v1/widget.js"
  data-title="Publikasi Terbaru"
  data-limit="5"
  data-category="politik"></script>

Alternatif tanpa JavaScript: umpan RSS di https://API-HOST/api/v1/feed.rss (dukung ?category=slug).
